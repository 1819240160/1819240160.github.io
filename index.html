记叙文

理解

  1. 理解题目：表层含义。深层含义
  2. 概括人物形象：1.形象概括；2.情节；3.引用原文
  3. 概括情节：1.找线索；2.按某种结构对等划分；3.辨明时地人事情等诸多要素概括；4.当有多个不同事件时，找共性概括（分析内在线索）
  4. 事件：一个动作即为一个事件；情节：由两个及两个以上的具有因果联系的事件构成情节

分析

思路：具体的描写或叙述 => 句子内容、结构、段落内容的补充与推动 => 形象的刻画和主题的突出

  1. 题目的作用：

    1. 时间：表明故事时间
    2. 地点。交代故事发生地点
    3. 时间地点统称环境
    4. 人、物：交代文章写作对象
    5. 时地人事物都能成为外在线索
    6. 时地人事物都有可能成为文章写作对象
    7. 事件则交代文章内容，两个及以上的事件可能构成悬念
    8. 情感类词汇暗示文章主旨
    9. 吸引读者兴趣（通用，不能单独成点）
    10. 特殊疑问词等，说明哪个是特殊疑问词及其效果（如倒装句）
    11. 如果题目带有修辞也要答出

  2. 环境描写的作用：1. 暗示人物命运；2. 设置人物活动背景；3.烘托人物心情，刻画人物形象；4. 为后文作铺垫；5.渲染气氛；6.推动故事情节发展

  3. 人物描写作用：1.作为文章线索贯穿全文；2.故事的参与者，交代故事内容，推动情节发展；3.人物形象的对比刻画；4.人物的精神品质和映衬主体

  4. 情节、倒叙和插叙的作用

    作用 情节 插叙 倒叙
    对主要情节做必要补充，推动情节发展，使情节更完整，内容更充实 √ √ √
    对人物的描写、衬托或刻画 √ √
    突出，深化文章主题 √ √ √
    使文章的行文结构富于变化，避免呆板的平铺直叙 √ √
    引发悬念 √
    吸引读者兴趣 √

  5. 句子作用

    结构或内容 句子位置 作用
    结构上 开头 1.总领全文，引出下文；2.设置悬念，吸引读者兴趣；3.交代背景，埋下伏笔或作铺垫
    中间 承上启下
    结尾 1.总结全文；2.首尾呼应
    内容上 1.表达感情，塑造人物；2.画龙点睛。升华主题；3.交代内容

  6. 小标题的作用：1.可用题目的作用（以分析题目的角度分析）；2.串联全文，使文章结构严谨

  7. 引用的作用：1.增强文章说服力；2.增添文章文采；3.突出人物形象；4.若引用的是情节，则有a.情节的补充与推动b.主题的突出c.作者的态度；5.增强文章的真实性

  8. 夸张的作用：1.突出事物的本质特征；2.表达作者的情感；3.烘托气氛，引起联想

  9. 对比的作用：1.使作品主题更突出；2.使人物形象更鲜明；3.使思想感情更深刻

赏析：

  1. 赏析题目：1.表层含义；2.深层含义；3.作用
  2. 赏析关键字：
    1. 修辞或描写
    2. 语境（结合句子内容对修辞或描写进行说明）
    3. 场景（要答出句子的作用，通常有设悬念、引出下文、首尾呼应、形成对比等）
    4. 情感（品质、特点）

概念

  1. 描写

    • 描写
      • 心里描写
        • 直接心里描写：内心独白、直感、幻觉描写
        • 间接心里描写：语言、神态、动作、环境等
      • 肖像描写
      • 细节描写
        • 作用：抓住事物特征，着重刻画，入微地再现特点
      • 白描
      • 语言、神态、动作、心里描写等

  2. 表达技巧：

    伏笔 ==必然==的，为引出后文所设下的情节
    铺垫 ==可选择性==的，为引出后文所设立的情节
    渲染 ==正面==描写
    烘托 ==侧面==描写
    联想 由一事物==想到==另一个==存在==的事物（真实的，存在的）
    想象 由一事物==加工出==另一个==不存在==的事物（虚假的，加工的）
    虚实结合 真实的具体描写和抽象的述说结合起来
    工笔细描 用细腻的笔触，精细地描绘，抓住事物的特征，浓墨重彩，着意刻画，入微地再现事物的特征，是细节描写的一种，但更追求工整
    正衬 以==好的==衬托好的
    反衬 以==不好的==衬托好的
    欲扬先抑 是一种写作手法，“扬”是指褒扬，“抑”是指贬低。褒扬某人或某物不从褒扬处落笔，而是先贬低，从相反的贬抑处落笔
    象征 一切形式对思想，有形对无形的约定俗成的表现1.具有普遍意义的2.将象征隐藏在作品特定的语言环境中来表达作者的潜在意图
    托物言志 用物来表达感情（象征）
    悬念 因果联想脱节而产生吸引读者兴趣的情节
    其他 开门见山、承上启下、首尾呼应、卒章显志、直抒胸臆、融情于景、情景交融、以乐景写哀情

  3. 修辞

    比喻 把一个事物比作另一个事物
    夸张 对客观的人或事物进行扩大、缩小或超前的描述
    反复 为突出某个意识或强调某种感情，刻意重复某个词语或句子
    通感 多感官相结合
    其他 排比、拟人、拟物、设问、反问、借代、引用、互文、起兴、比兴

分类

  1. 认知能力
    1. 理解（转述或概括）
    2. 分析（部分与部分之间的内在联系）
    3. 赏析（以某种标准进行判断）
    4. 创造（把某些要素按某种联系或目的串联起来）
  2. 表达方式（篇章）
    • 表达方式：（篇章）
      • 说明
      • 记叙
        • 事件、情节、文章、时间
        • 外在线索：人、事、物
        • 内在主题，内在联系（主旨）
      • 议论
        • 论点、论据、论证
      • 描写
        • 动作、语言、神态、心理、环境、肖像
      • 抒情
        • 直接抒情（含有情感类的词汇）
        • 间接抒情（与景物相结合）
  3. 表达技巧（段落句群）
    记叙 悬念、伏笔、照应、象征、联想
    描写 想象、渲染、烘托、对比、衬托
    动静结合、色味结合、虚实结合、正侧结合
    抒情 借景抒情、融情于景、以乐景写哀情、直抒胸臆
  4. 修辞手法（句子，句群）：比喻、拟人、排比、夸张、对偶、设问、反问、反复等

其他：

  1. 第一人称叙述的好处：1。代入感强，使读者身临其境；2.便于抒情；3.叙述亲切自然
  2. 第三人称叙述的好处：1.叙述中立客观；2.增强故事的可信度
  3. 礼貌原则：1.自己多吃亏，别人多受益；2.谦虚；3.一致观点，肯定他人观点，尽量不违背他人观点；4.同情
  4. 四表（综合性学习）：1.表礼貌；2.表身份；3.表来意；4.表感谢

说明文

说明文的本质：客体—>特征
只要出现了事物特征就一定出现说明对象
  1. 说明方法：为了把事物的特征，形状，功能，性质等介绍清楚而采用的一些表达方式
    名称 含义 答题模式
    列数字 用具体的数字来说明事物的外形或其他特点 用数字具体而准确地说明了事物的xx特点，使说明更准确，更有说服力
    举例子 为了说明输完的某一特点 用数据具体而说明了事物的xx特点，从而使说明更准确，更有说服力
    作比较 将两种事物相同或不同的事物现象等加以说明 把xx和xx相比较，突出强调了被说明对象的xx特点（地位、影响等），让人印象鲜明，深刻
    打比方 通过比喻来说明事物的特征 把xx比作xx，生动形象地说明了事物的xx特征，增强了文章的趣味性
    分类别 根据事物的形状、性质、成因、功能等把事物分成若干类，依照类别加以说明 对事物分门别类加以说明，使说明更有条理性
    摹状貌 通过描写具体事物的具体形状或面貌来说明事物的特征（先满足其他说明方法，再满足摹状貌）
    画图表 借助表格，插图，照片等加以说明 运用画图表的说明方法，对事物的xx特征加以说明，使说明更简明，条理，直观
    引资料 引用一些文献资料、诗词、俗语、名人名言等来加以说明事物特征 引用了xx资料，说明了事物的xx特征，使说明更生动形象（文学性文本）或更具体准确（新闻、数据等）
    作诠释 对事物或事理的某些性质或特征作适当的分析说明（可以作为人或物的征象、标志等，特点：人或事物所具有的独特特征）
    下定义 用简洁明确的语言对事物的本质特征作概括说明（属+种差或全部列举）
  2. 说明文特点：1。语言严谨，准确2.思维的条理性和严密性
  3. 体现语言严谨性的答题方式：
    题目类型 答题模式
    词语在句子中的作用 1.解词（+限制类型）；2.结合句中说明对象解释词语；3.体现说明文语言的严谨准确性
    句子中的词能否删去 1.不能；2.解词（+限制类型）；3.结合句中说明对象解释词语；4.==删去==后意思会变成xx（不符合事实）；5.该词体现说明文语言的严谨准确性
    能否替换某词语 1.不能；2.解词（+限制类型）；3.结合句中说明对象解释词语；4.==替换==后意思会变成xx（不符合事实）；5.该词体现说明文语言的严谨准确性
    分析句子的语言特点 1.找词；2.解词（+限制类型）；3.结合句中说明对象解释词语；4.该词体现说明文语言的严谨准确性
  4. 体现说明文语言准确性的词语（各种限定词）
    词性 限制类型 例子
    形容词 性质 好的、坏的
    状态、情态 高、低、长、短
    副词 表时间、频率 刚刚、当时、常常
    表方位、处所 这里、那里
    表程度 最、几乎
    表肯定或否定 一定、的确、绝对、绝不
    表估计或推测 一般、大约、可能
    表范围 一切、部分、所有
  5. 体现生动性的答题方式：1.修辞；2.表达方式；3.表现（写作）手法；4.句式变化；5.说明方法
