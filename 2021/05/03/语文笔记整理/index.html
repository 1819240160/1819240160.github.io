语文笔记整理

记叙文

理解

  1. 理解题目：表层含义。深层含义
  2. 概括人物形象：1.形象概括2.情节3.引用原文
  3. 概括情节：1.找线索2.按某种结构对等划分3.辨明时地人事情等诸多要素概括4.当有多个不同事件时，找共性概括（分析内在线索）
  4. 事件：一个动作即为一个事件；情节：由两个及两个以上的具有因果联系的事件构成情节

分析

思路：具体的描写或叙述 => 句子内容、结构、段落内容的补充与推动 => 形象的刻画和主题的突出

  1. 题目的作用：

    1. 时间：表明故事时间
    2. 地点。交代故事发生地点
    3. 时间地点统称环境
    4. 人、物：交代文章写作对象
    5. 时地人事物都能成为外在线索
    6. 时地人事物都有可能成为文章写作对象
    7. 事件则交代文章内容，两个及以上的事件可能构成悬念
    8. 情感类词汇暗示文章主旨
    9. 吸引读者兴趣（通用，不能单独成点）
    10. 特殊疑问词等，说明哪个是特殊疑问词及其效果（如倒装句）
    11. 如果题目带有修辞也要答出

  2. 环境描写的作用：1. 暗示人物命运 2. 设置人物活动背景 3.烘托人物心情，刻画人物形象 4. 为后文作铺垫5.渲染气氛6.推动故事情节发展

  3. 人物描写作用：1.作为文章线索贯穿全文2.故事的参与者，交代故事内容，推动情节发展3.人物形象的对比刻画4.人物的精神品质和映衬主体

  4. 情节、倒叙和插叙的作用

    作用 情节 插叙 倒叙
    1. 对主要情节做必要补充，推动情节发展，使情节更完整，内容更充实 √ √ √
    2. 对人物的描写、衬托或刻画 √ √
    3. 突出，深化文章主题 √ √ √
    4. 使文章的行文结构富于变化，避免呆板的平铺直叙 √ √
    5. 引发悬念 √
    6. 吸引读者兴趣 √

  5. 句子作用

    结构或内容 句子位置 作用
    结构上 开头 1.总领全文，引出下文2.设置悬念，吸引读者兴趣3.交代背景，埋下伏笔或作铺垫
    中间 承上启下
    结尾 1.总结全文2.首尾呼应
    内容上 1.表达感情，塑造人物2.画龙点睛。升华主题3.交代内容

  6. 小标题的作用：1.可用题目的作用（以分析题目的角度分析）2.串联全文，使文章结构严谨

  7. 引用的作用：1.增强文章说服力2.增添文章文采3.突出人物形象4.若引用的是情节，则有a.情节的补充与推动b.主题的突出c.作者的态度5.增强文章的真实性

  8. 夸张的作用：1.突出事物的本质特征2.表达作者的情感3.烘托气氛，引起联想

  9. 对比的作用：1。使作品主题更突出2.使人物形象更鲜明3.使思想感情更深刻

赏析：

  1. 赏析题目：1.表层含义 2.深层含义 3.作用
  2. 赏析关键字：
    1. 修辞或描写
    2. 语境（结合句子内容对修辞或描写进行说明）
    3. 场景（要答出句子的作用，通常有设悬念、引出下文、首尾呼应、形成对比等）
    4. 情感（品质、特点）

概念

  1. 表达方式：
    • 表达方式：说明、记叙、议论、抒情、说明
      • 心里描写
        • 直接心里描写：内心独白、直感、幻觉描写
        • 间接心里描写：语言、神态、动作、环境等
      • 肖像描写
      • 细节描写
        • 作用：抓住事物特征，着重刻画，入微地再现特点
      • 白描
      • 语言、神态、动作、心里描写等
  2. 表达技巧：
    伏笔 必然的，为引出后文所设下的情节
    铺垫 可选择性的，为引出后文所设立的情节
    渲染 正面描写
    烘托 侧面描写
    联想 由一事物想到另一个存在的事物（真实的，存在的）
    想象 由一事物加工出另一个不存在的事物（虚假的，加工的）

其他：

  1. 第一人称叙述的好处：1。代入感强，使读者身临其境2.便于抒情3.叙述亲切自然
  2. 第三人称叙述的好处：1.叙述中立客观2.增强故事的可信度
  3. 礼貌原则：1.自己多吃亏，别人多受益2.谦虚3.一致观点，肯定他人观点，尽量不违背他人观点4.同情
  4. 四表（综合性学习）：1.表礼貌2.表身份3.表来意4.表感谢
